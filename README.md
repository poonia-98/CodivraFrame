# CodivraFrame


CodivraFrame is a modern website layout generator that helps users instantly create professional business websites.  
Users can select a business type, customize branding, preview a premium layout, and download clean, editable HTML, CSS, and JavaScript code.


# Features

- Generate websites for multiple business types  
  (Cafe, Restaurant, Gym, Salon, Portfolio, Local Business)
- Premium hero sections with dynamic background images
- Always-readable text using smart image overlays
- Clean, minimal, and modern UI
- Download only the generated website code
- No frameworks â€” pure HTML, CSS, and JavaScript



# Built With

- HTML5  
- CSS3  
- Vanilla JavaScript  

No libraries. No dependencies.



# How It Works

1. Enter your brand or company name  
2. Select a business type  
3. Preview the generated website layout  
4. Download the source code  
5. Customize and deploy anywhere
